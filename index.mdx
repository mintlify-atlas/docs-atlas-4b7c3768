title: "Incidencias Backend: Civic Incident Reporting Platform"
description: "Incidencias Backend is a Laravel API powering a mobile app for citizens to report civic incidents and an admin panel for managing those reports."
---

Incidencias Backend is a full-stack civic reporting platform built on Laravel 12. Citizens submit geo-located incident reports through a mobile app, and administrators manage and resolve those reports through a web panel.
